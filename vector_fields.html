You can use the following base code to plot and experiment with vector fields

#2D vector fields
x,y=var('x y')         #define the variables
f(x,y) = (x^2,y^2)     #define the 2D vector field
v1 = plot_vector_field(f,(x,-5,5),(y,-5,5))     #store the plot of the vector field
show(v1)      #show the vector field
Messages 

#3D vector fields
x,y,z=var('x y z')         #define the variables
f(x,y,z) = ( x^2 , y^2 , z^2 )       #define the 3D vector field
v1 = plot_vector_field3d(f,(x,-5,5),(y,-5,5),(z,-5,5))     #store the plot of the vector field
show(v1)      #show the vector field
Messages 

#2D Gradient vector field, contours, and integral curves
x,y = var('x y')
f(x,y) = x^2/4 + y^2/9  #scalar field
p1 = implicit_plot(f==1, (x,-10,10),(y,-10,10))      #store a plot of the k=1 level curve
p2 = implicit_plot(f==2, (x,-10,10),(y,-10,10),color='red')  #store a plot of the k=2 level curve
p3 = implicit_plot(f(x,y)==4, (x,-10,10),(y,-10,10),color='green')  #store a plot of the k=4 level curve
p4 = implicit_plot(f(x,y)==9, (x,-10,10),(y,-10,10),color='gold')  #store a plot of the k=9 level curve

grad_f = (f.diff(x),f.diff(y))   #Compute the gradient of f
v1 = plot_vector_field(grad_f,(x,-10,10),(y,-10,10))  #store a plot of the vector field
p1+p2+p3+p4+v1      #Graph all of the things in one plot
Messages

Graphing the integral curves is a lot more involved (must solve an ODE)

y = function('y')
DE = diff(y(x),x)==4/9*y(x)/x
sols = [desolve(DE,[y(x),x],ics=[1,i]) for i in [1,0,-1]]
p1+p2+p3+p4+v1+plot(sols,x,-10,10)
